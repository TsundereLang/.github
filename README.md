## About

Tsundere is a modern programming language and ecosystem built specifically for Discord development.

Our goal is to provide a cleaner, safer, and more powerful developer experience while remaining fully compatible with the JavaScript ecosystem.

The language uses the `.yuri` file extension and compiles to JavaScript, allowing developers to leverage the Node.js ecosystem while benefiting from Tsundere's tooling, abstractions, and Discord-first workflow.

---

## Core Features

* Discord-first development experience
* `.yuri` source files
* JavaScript interoperability
* Strong IntelliSense support
* Type-safe Discord APIs
* Advanced editor tooling
* Modern language design
* Automatic Discord type generation
* YuriLS language server
* VS Code integration
